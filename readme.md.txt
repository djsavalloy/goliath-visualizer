# 📡 DJ Savalloy Audio Grid | Project Goliath

**A Sovereign Digital Node for Real-Time Audio Visualization.**

This repository contains the source code for the **DJ Savalloy Audio Grid**, a high-fidelity, audio-reactive Progressive Web App (PWA). Built as part of the digital infrastructure for **Project Goliath CIC**, this tool operates as a visual feedback mechanism for vocational audio engineering training.

## 🏗️ The Technical Payload

* **Real-Time Spectrum Analysis:** 64-band frequency rendering utilizing HTML5 Canvas and the Web Audio API.
* **Dual-Wave Oscilloscope:** Real-time time-domain waveform tracking mapping high and low frequencies.
* **Dynamic Bass-Pulse Engine:** Central visual node scales dynamically based on low-end frequency input.
* **PWA Architecture:** Fully installable as a standalone application on iOS, Android, and Desktop environments with offline-caching capabilities.
* **Ignition Switch Protocol:** Built-in manual start overlay to comply with modern browser auto-play and microphone security restrictions.

## ⚙️ The Materials (Tech Stack)
* **Structure:** HTML5
* **Styling:** CSS3 (Flexbox, Fixed Positioning, Dynamic Shadows)
* **Logic:** Vanilla JavaScript (ES6+)
* **Engine:** Web Audio API (`AudioContext`, `AnalyserNode`)
* **Deployment:** Service Workers (`sw.js`) & Web App Manifest (`manifest.json`)

## 🚀 The Load-Path (Installation)

Because this application is built as a PWA, it can be deployed directly to any mobile device or desktop without the need for traditional App Store routing.

1. Navigate to the live hosted URL (via GitHub Pages).
2. **On iOS/Safari:** Tap 'Share' > 'Add to Home Screen'.
3. **On Android/Chrome:** Tap the menu (three dots) > 'Install App'.
4. **On Desktop:** Click the install icon in the URL address bar.
5. Launch the application and click **INITIALIZE AUDIO GRID** to grant microphone access.
